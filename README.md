## 🚀 Getting Started

Clone the project

```bash
  git clone https://github.com/rizzkiardi/TokoBit.git
```

Go to the project directory

```bash
  cd TokoBit
```

Open in browser

```bash
  http://127.0.0.1:5500/index.html
```
